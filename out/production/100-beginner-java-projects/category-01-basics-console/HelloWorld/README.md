# Hello World

A simple Java project demonstrating the basics of Object-Oriented Programming (OOP).

## Features

* Create and use classes
* Create objects
* Call methods
* Print output to the console

## Files

| File            | Description                               |
| --------------- | ----------------------------------------- |
| Main.java       | Program entry point                       |
| HelloWorld.java | Contains the HelloWorld class and methods |

## How to Run

Compile:

```bash
javac *.java
```

Run:

```bash
java Main
```

## Example Output

```text
Hello, World!
```

## Technologies

* Java
* Console Application
* Object-Oriented Programming (OOP)
